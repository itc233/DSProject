## Usage
1. Set up GOPATH

    `export GOPATH=$YOUR_PATH_TO_DSProject/DSProject`

1. Build web application

    `go build main.go`

1. Serve the application

    `./main -port your_port`

    See more instrucitons by
  
    `./main -h`
